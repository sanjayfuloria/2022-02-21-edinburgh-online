Noteable service

For this workshop, we will be using the University of Edinburgh Noteable service. To prepare, please make sure you can log in with your EASE credentials and set up new notebooks as described below.

  1. In your browser, go to the University of Edinburgh Noteable login page and use your EASE credentials to access the site.

  2. When prompted to select a personal notebook server, choose ‘Standard Notebook (Python 3)’ and select ‘Start’.

  3. You should see a Jupyter Notebook screen. Select the ‘New’ dropdown menu at the top right and choose ‘Folder’.

  4. The directory listing will show ‘Untitled Folder’. Tick the box to the left of it. A ‘Rename’ button will appear above the listing; select it.

  5. A dialogue box appears, asking you to enter a new directory name. Type ‘Desktop’ and select ‘Rename’.

  6. Select the ‘Desktop’ folder to enter it.

  7. Using the ‘Upload’ button on the top right, upload your downloaded copies of shell-lesson-data.zip and python-novice-gapminder-data.zip. You will need to select another ‘Upload’ button in the directory listing each time.

  8. Select the ‘New’ dropdown menu at the top right and choose ‘Terminal’.

  9. In the new tab that opens, you should see a black Terminal window. Enter the following text, one line at a time.

    cd Desktop
unzip shell-lesson-data.zip
mkdir python-lesson
unzip python-novice-gapminder-data.zip -d python-lesson
exit

  10. You can now close the Terminal tab. In your original tab, you should see your new shell-lesson-data and python-lesson folders. You can select and delete the zip files if you like, by ticking the boxes next to them and selecting the red dustbin icon above.
